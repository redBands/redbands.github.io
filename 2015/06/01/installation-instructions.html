Octopress Three Installation Log

  • Create a new custom workspace on C9 and name it.
  • install jekyll*
  • install octopress3
  • install octopress-ink*
  • name new blog
  • change into the directory for the new blog
  • initialise the deploy setup

Git instructions

  • initialise git: git init
  • link to repository: git remote add origin git@github.com:redBands/redbands.github.io.git
  • add files: git add .
  • add commit message: git commit -m 'octopress 3 install'
  • change branch: git checkout -b source
  • push files: git push -u origin source

Carry on instructions

  • build website: bundle exec jekyll build
  • deploy website: octopress deploy

  • start preview server: jekyll serve –host $IP –port $PORT –baseurl
